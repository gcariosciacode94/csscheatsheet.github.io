CSS Properties Cheat Sheet

Quick Reference for Basic CSS

Property Description Example
color Sets the color of text. color: red;
background-color Sets the background color of an element. background-color: blue;
background-image Sets the background of an element to an image. background-image: url("/shared-assets/images/examples/lizard.png");
margin Create space around an element. margin: 25px;
padding Add space inside an element. padding: 15px;
border Customize the border around an element. border: 2px solid red;
border-radius Create rounded corners. border-radius: 5px;
box-shadow Create a shadow effect. box-shadow: 3px 3px 5px 1px rgba(0, 0, 0, 0.4);
font-family Styles the font. font-family: Helvetica;
font-size Sets the size of the font. font-size: 16px;
font-weight Sets the thickness of the font. font-weight: 700;
